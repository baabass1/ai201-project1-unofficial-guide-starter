Python File Handling

Python can create, read, write, and modify files.

Opening a file:

file = open("example.txt", "r")

Reading a file:

content = file.read()

Closing a file:

file.close()

A safer approach uses the with statement.

Example:

with open("example.txt", "r") as file:
content = file.read()

Common file modes include:

"r" for reading

"w" for writing

"a" for appending

File handling is important for storing data, reading datasets, and saving program output.
